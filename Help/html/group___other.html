Рисуем Яблоко (может упасть с дерева)

Заметки
Яблоки они такие, не знаешь когла и на кого упадёт...
Аргументы
x- x координата центра Яблока
y- y координата центра Яблока
size- размер Яблока. Если параметр равен 1 (это размер как на дереве PaintAppleTree), размер Яблока 20 на 20 пикселей
color- цвет облака
Пример использования
for (int t = 0; t <= 200; t += 10)
{
PaintBackground();
PaintAppl (820 - (int) (t*0.2), 300 + t, 1); // Яблоко падает с Яблони, катится к нам и увеличивается (так как становится ближе)
t++;
txSleep (50);
}
