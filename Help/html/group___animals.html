Рисуем Котика

Аргументы
x- x координата точки где сидит Котик
y- y координата точки где сидит Котик
sizeX- размер Котика по ширине. Если параметр равен 1 (это норма), размер Котка примерно 120 на 120 пикселей
sizeY- размер Котика по высоте. Если параметр равен 1 (это норма), размер Котка примерно 120 на 120 пикселей
nose- параметр позволяет двигать нос (и усы) вверх и вниз. Если параметр равен 0 (это норма). К примеру, если параметр "+5" - то нос ниже, если параметр "-5" - то нос выше обычного положения
glazCatZrachek- параметр позволяет двигать зрачки котика влево и вправо. К примеру, если параметр "+5" - то зрачки правее, если параметр "-5" - то зрачки левее центра глаза.
sizeGlazCat- размер глаз Котика. Если параметр равен 0 (это норма), параметр дает возможность Котику моргать, прищуриться или вытаращить глаза
LeftYho- параметр двигает левое ухо.
RightYho- параметр двигает правое ухо.
LeftNogaCat- расположение левой лапы Котика. Если парамерт равен 0, то Котик стоит (не шагает)
RightNogaCat- расположение правой лапы Котика. Если парамерт равен 0, то Котик стоит (не шагает)
khvost- хвост Котика. "0" - стандартное состояние. Хвостом можно двигать
color- задает цвет кота. Параметр позволяет создавать котиков разного цвета.
Пример использования
PaintCat (60, 120, 1, 1, 2, 5, 0, 0, 0, 1, 1, -20, ryzhiy);

PaintCat (750 + (int) (0.5*t), 440, 0.7, 0.9, 1, 1, 1, (-t/10) % 10, (t/10) % 10, ((t/10) % 2*2-1)* 5, (t/10) % 2* (-5), 0, ryzhiy);
PaintCat (250 + (int) (0.5*t), 510 - (int) (0.2*t), 1.2, 1.2, 1, 1, 1, (-t/5) % 10, (t/5) % 10, ((t/10) % 2*2-1)*10, (t/10) % 2*(-10), ((t/10) %2 * 2 - 1)*30, brown);
t++;
txSleep (200);
}

Рисуем Цыпленка

Заметки
Цыплята гуляют в саду. Моя версия отрисовки цыпленка, у тебя может получиться лучше ;)
Аргументы
x- x координата центра тела Цыпленка
y- y координата центра тела Цыпленка
size- размер Цыпленка. Если параметр равен 1 (это норма), размер Цыпленка примерно 40 на 45 пикселей
glazCipa- размер глаза Цыпленка. Если параметр равен 1 (это норма), параметр дает возможность Цыпленку прищуриться или вытаращить глаза
LeftNogaCipa- расположение левой ноги Цыпленка. Если парамерт равен 0, то Цыпленок стоит (не шагает)
RightNogaCipa- расположение правой ноги Цыпленка. Если парамерт равен 0, то Цыпленок стоит (не шагает)
LR- направление в которое повернут Цыпленок. "1" - Цыпленок смотрит направо, "-1" - Цыпленок смотрит налево
OpenKluv- клюв Цыпленка можно открыть или закрыть. "0" - клюв закрыт, "10" - клюв открыт широко. Желательно менять значения от 0 до 10, но это не обязательно.
Пример использования
PaintCipa (25, 25, 1, 1, 0, 0, 1, 10); // цыпленок размера 40 на 45 пикселей, спокойно стоит, смотрит вправо, но у него открыт клюв.
PaintCipa (10 + 2*t, 600 + 0.1*t, 0.8, 1, ((t/10) % 2*2-1)*5, (t/10) % 2*(-5), t%2*2 - 1, t%2 * 10); //цыпленок меньше своего стандартного размера, шагает по диагонали, смотрит в разные стороны и разговаривает

Рисуем Куру

Заметки
Цыплята гуляют в саду под присмотром своей мамы. Моя версия Курочки
Аргументы
x- x координата центра тела Куры
y- y координата центра тела Куры
size- размер Куры. Если параметр равен 1 (это норма), размер Куры примерно 110 на 110 пикселей
glazKura- размер глаза Куры. Если параметр равен 1 (это норма), параметр дает возможность Курочке моргать, прищуриться или вытаращить глаза
LeftNogaKura- расположение левой ноги Куры. Если парамерт равен 0, то Кура стоит (не шагает)
RightNogaKura- расположение правой ноги Куры. Если парамерт равен 0, то Кура стоит (не шагает)
LR- направление в которое повернута Кура. "1" - Кура смотрит налево, "-1" - Кура смотрит направо
OpenKluv- клюв Куры можно открыть или закрыть. "0" - клюв закрыт, "10" - клюв открыт широко. Желательно менять значения от 0 до 10, но это не обязательно.
flap- крыло Куры, крылом можно махать. "0" - спокойное состояние, крыло сложено
Пример использования
PaintKura (60, 60, 1, 1, 0, 0, 1, 0, 0); // Кура размера 110 на 110 пикселей, спокойно стоит, смотрит налево, клюв закрыт.
