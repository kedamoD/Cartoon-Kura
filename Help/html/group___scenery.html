Рисуем дерево Ель

Заметки
Цыплята гуляют в саду, в котором есть разные деревья, в томчисле и ёлочки
Аргументы
x- x координата верхней части ствола ёлки (центр рисунка ёлки)
y- y координата верхней части ствола ёлки (центр рисунка ёлки)
sizeX- размер ёлки по ширине. Если параметр равен 1, то ширина ёлки примерно 180 пикселей
sizeY- размер ёлки по высоте. Если параметр равен 1, то высота ёлки примерно 180 пикселей
veter- ветер качает ёлку ("0" - нет ветра; "-20" - наклон налево; "+30" - наклон направо)
Пример использования
PaintSpruce (100, 150, 1, 1, 0);
PaintSpruce (500, 300, 0.5, 0.8, -20);
PaintSpruce (700, 270, 1.3, 1.4, +30);
